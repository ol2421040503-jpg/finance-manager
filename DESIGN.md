# DESIGN.md

## 气质与意象
高端私人银行客户端的沉静感——深色胡桃木书桌上摊开账本，午后自然光透过百叶窗洒在数字屏幕上。不是冰冷的金融科技风，而是温暖、可信赖、有人情味的财务管理空间。销售人员的钱袋子，既要严谨又要让人安心。

## 配色方案
- **主背景**: 暖白 `#FAFAF8`（米白纸张感）/ 深色模式 `#0F1117`
- **卡片/面板**: 纯白 `#FFFFFF` / 深色 `#1A1D27`
- **主强调色**: 琥珀金 `#B8860B` → Tailwind amber-700 系列，代表财富与稳健
- **次要色**: 石板灰 `#475569`，用于正文与辅助信息
- **语义色**: 
  - 收入/增长: emerald-600 `#059669`
  - 支出/扣减: rose-600 `#E11D48`
  - 警告/不足: amber-500 `#F59E0B`
  - 安全/满额: sky-600 `#0284C7`
- **边框**: `#E2E8F0`（浅灰分隔线）

## 字体排版
- 中文: 系统默认 `PingFang SC` / `Microsoft YaHei`
- 金额数字: `font-variant-numeric: tabular-nums`，等宽对齐
- 标题: 18-20px semibold
- 金额: 24-32px bold，作为视觉焦点
- 正文: 14px regular

## 页面结构
- 左侧固定侧边栏导航（可折叠），右侧主内容区
- 侧边栏: 深色底，琥珀金高亮当前页
- 每个功能页顶部有简短说明文字
- 仪表盘页: 7个账户卡片网格 + 总资产 + 资金流向概览

## 组件规范
- 卡片: 圆角 `rounded-xl`，阴影 `shadow-sm`，内边距 `p-6`
- 表格: 紧凑行高，偶数行微灰底
- 表单: 左对齐标签 + 输入框，下方灰色提示文字
- 按钮: 主操作用 amber 实色，次要操作用 outline 样式
- 金额: 右对齐，千分位格式，保留两位小数

## 动效与交互
- 页面切换: 淡入 `animate-in fade-in`
- 卡片悬停: 微上浮 `hover:-translate-y-0.5 hover:shadow-md`
- 数据变更: 数字变化时短暂高亮
- 对话框: 从中心缩放弹出

## 设计禁忌
- 禁止使用科技蓝/紫色渐变（太冷、太模板化）
- 禁止卡片使用发光边框或霓虹效果
- 禁止大段纯文字无视觉分隔
- 禁止金额左对齐（必须右对齐或居中）
- 禁止表格行高过大导致信息密度降低
