# DESIGN.md

## 气质与意象
高端私人银行客户端的沉静感——深色胡桃木书桌上摊开账本，午后自然光透过百叶窗洒在数字屏幕上。不是冰冷的金融科技风，而是温暖、可信赖、有人情味的财务管理空间。销售人员的钱袋子，既要严谨又要让人安心。

## 配色方案
- **主背景**: 暖白 `#FAFAF8`（米白纸张感）/ 深色模式 `#0F1117`
- **卡片/面板**: 纯白 `#FFFFFF` / 深色 `#1A1D27`
- **主强调色**: 琥珀金 `#B8860B` → Tailwind amber-700 系列，代表财富与稳健
- **次要色**: 石板灰 `#475569`，用于正文与辅助信息
- **语义色**: 
  - 收入/增长: emerald-600 `#059669`
  - 支出/扣减: rose-600 `#E11D48`
  - 警告/不足: amber-500 `#F59E0B`
  - 安全/满额: sky-600 `#0284C7`
- **边框**: `#E2E8F0`（浅灰分隔线）

## 字体排版
- 中文: 系统默认 `-apple-system` / `PingFang SC` / `Microsoft YaHei`
- 金额数字: `font-variant-numeric: tabular-nums`，等宽对齐
- 页面标题: 20px semibold
- 金额: 24-28px bold，作为视觉焦点
- 正文: 12-13px regular（移动端紧凑）

## 页面结构（移动端优先）
- 顶部标题栏（12px 琥珀金 logo + 标题）
- 底部 5 个 Tab 导航：总览、提成、其他收入、支出、更多
- 「更多」弹出底部 Sheet，含基础信息、存量分配、账户余额、综合历史、规则说明
- 每个功能页顶部有简短说明文字
- 仪表盘: 紧凑列表式账户卡片 + 总资产卡片
- 最大内容宽度 `max-w-lg` 居中

## 移动端适配
- iOS safe area: 顶部刘海 + 底部 Home Indicator 适配
- 触摸优化: 最小点击区域 44px，输入框高度 40px
- 表格 → 可折叠卡片列表（点击展开详情）
- 表单全宽纵向排列，2 列网格仅用于年份/月份选择
- 按钮全宽 `w-full`，高度 44px
- 禁止弹性滚动 `overscroll-behavior: none`
- 隐藏滚动条但保留滚动功能
- 输入框去除 number 类型的 spinner
- `touch-action: manipulation` 禁止双击缩放

## 组件规范
- 卡片: 圆角 `rounded-xl`，内边距 `p-3`（移动端紧凑）
- 表单: 全宽输入框 + 上方标签 + 下方灰色提示
- 按钮: 主操作用 amber 实色全宽，次要操作用 outline
- 金额: 右对齐，千分位格式，保留两位小数
- Tab 导航: 底部固定，半透明毛玻璃背景

## 动效与交互
- 按钮点击: `active:scale-95` 缩放反馈
- 列表展开/折叠: 点击切换详情
- 底部 Sheet: 从底部滑入
- 对话框: 从中心缩放弹出

## 设计禁忌
- 禁止使用科技蓝/紫色渐变（太冷、太模板化）
- 禁止卡片使用发光边框或霓虹效果
- 禁止大段纯文字无视觉分隔
- 禁止金额左对齐（必须右对齐或居中）
- 禁止桌面端宽表格（移动端用卡片列表替代）
